# mymgm-twitter-bot

a [Sails](http://sailsjs.org) application

## Install Instructions:

Clone this repo

Run: npm install

## How to run the app

To start it:

cd <checkout-location>
sails lift

Then go to:
http://localhost:1337/

Then to save twitter token, go to:
http://localhost:1337/tweetbot/getTwitterRequestToken
